Editing a Landscape to Develop Treatment Alternatives

Landscape Editing can occur in both the Landscape Evaluation stage of the Planning Cycle under Landscape Edit, or within the Strategic Planning stage under Develop Treatment Alternatives. Menu options and results are the same for both. In editing sessions accessed from the Landscape Evaluation Section, any landscape can be edited. In editing sessions accessed from the Develop Treatment Alternatives section only certain landscapes may be selected to ensure the comparison between landscapes of the same extent.

In Developing Treatment Alternatives, your Originating Landscape must be an Extracted, unedited LANDFIRE landscape and the only landscapes available for you to edit must fall within the same landscape extent.

Getting Ready to Edit

Edit Landscape will walk you through editing your landscape based on a default set of rules you select, or a custom set of rules you create.

First, select your Originating Landscape to edit from the dropdown menu.

Notice you may access the Landscape rules by either clicking on the + icons under the landscape dropdown menu, or select the dropdown headings . As you select or create rules, they will be listed in the Edit Landscape section of the page.

To split your display to enable a map view of your landscape on the left side of the screen, and the landscape edit interface on the right, select the globe icon at the top left of the page.

You may choose to edit your landscape in two ways: By adding Default Fuel Treatment / Disturbance Edit Rules and/or by adding your own Landscape Edit Rules.

Add Default Fuels Treatment/Disturbance Rule

Default Fuel Treatment / Disturbance Edit Rules use previously existing LANDFIRE Lookup tables (LFLU) and Fuel Model Lookup (FMLU) tables. The LFLU tables were created in an effort to improve the LANDFIRE fuels layers between versions. The LFLU tables are used to adjust the fuel attributes of cells, including: fuel model, canopy cover, stand height, canopy base height, and canopy bulk density, based on the type and intensity of treatment or disturbance and the time that has elapsed since that treatment or disturbance. For more technical details on these rules, visit the Default Edit Rule Technical Documentation Topic.Here you will edit landscapes to develop your Existing Condition (EC) if needed, and your treatment alternatives. The only landscape(s) available to edit at this point will be landscape “0” and any landscapes matching that extent. Create your alternatives by altering your Existing Condition landscape, whether that’s the "0" landscape, an existing landscape that has been made available in this step, or one that you create in this editing step from landscape "0". Whatever the case might be, it is very important to have an Existing Condition to ensure more accurate comparison results to gauge how each alternative does or does not meet objectives. The Existing Condition may need to reflect new disturbances on your landscape, like recent wildfires, fuels treatments, or windfalls.The first step in applying a Default Fuel Treatment / Disturbance Edit Rule is to specify the Treatment or Disturbance Type, and when prompted, select intensity or severity of the fuel treatment or disturbance type.Next you will select the Time Since Treatment or Disturbance that is most appropriate.Lastly you will decide on the extent to which you will apply the rule selected by either selecting a Landscape Mask, which will limit the rule to that shape, or if no Landscape Mask is selected will apply it to the entire landscape.Landscape Edit Rules1 Year Since Disturbance:2-5 Years Since Disturbance:6-10 Year Since Disturbance:  You have the option of defining a Landscape Mask to apply your rules to. Once you have set your rule, select Add to Rules. You may add as many rules as necessary.Rules are applied top to bottom, first by Default Fuels Treatment / Disturbance Edit Rules then by user defined Landscape Edit Rules. Currently there is no way to order your rules other than by the order in which you enter them, so be deliberate about how you apply rules and in what order you enter them.

Add Landscape Edit Rule

You may create your own landscape edit rules by selecting specific conditions, and specifying what modifications to make where ever those conditions occur.

To split the view and display your mapped area on the left, and this interface on the right, select the show split map button at the top left of the page.

  1. Begin by selecting your Starting Landscape which you would like to edit
  2. View the Add Landscape Rule options by either selecting the Add Landscape Edit Rule + sign just below the starting landscape field, or select the dropdown arrow to the far right of the Add Landscape Edit Rule heading.
  3. Specify the conditions you would like to select in the fields below Where these conditions exist. In this example we have selected 'Fuel Model' as the Attribute, 'is equal to' as the Operator and fire behavior fuel model '165 (TU5)' as the Value. This selected all TU5 fuel modeled areas across the landscape.
  4. Specify the conditions you would like to change by populating the fields below Modify the following values. In this example we have selected 'Canopy Cover' as the Attribute, 'decrease by' as the Operator, and '5' (for 5 percent) as the Value. For all TU5 fuel modeled areas, we will decrease the canopy cover by 5 percent.
  5. The Apply to Landscape Mask (optional) dropdown allows you to select a shapefile to which these edits will be contained. If this is left blank, the edits will be applied to the entire landscape.
  6. When you are ready to apply this rule select the Add to Rules button. Notice the rule has been added to you list, and the order noted. You can delete rules before saving by using the delete button to the right of each rule.

    7. Rules are applied top to bottom, first by Default Fuels Treatment / Disturbance Edit Rules then by user defined Landscape Edit Rules. Currently there is no way to order your rules other than by the order in which you enter them, so be deliberate about how you apply rules and in what order you enter them.

    As rules are added, will see them listed at the to of the page, along with the order they have been added, and a notification to let you know the new rule has been successfully saved.

  7. Once you have all the rules, in the order you would like them applied, give your landscape a descriptive name and select Save New Landscape. Your original landscape will remain, and an edited copy will now be available throughout IFTDSS.
Applying Landscape Edits

Once your rules are selected, give the new edited landscape a descriptive name and select Create New Landscape.

Edited landscapes will be created as new files, this ensures you will always have the original landscape for comparison. You may repeat this process to create your Existing Condition landscape if you need one, and the treatment alternative landscapes.

Edited landscapes are available in My Workspace, stored by default in the same folder as the landscape you are making edits to.

When clicking on landscape files that have been edited in My Workspace, you will see an option to View Edit Rules. This displays the edits that were made on that landscape.
