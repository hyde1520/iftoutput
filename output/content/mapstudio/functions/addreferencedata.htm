Adding Reference Data

 

You may add reference data layers to Map Studio that outline agency boundaries, wildfire activity, and other information.

To add a reference layer....

To add a reference layer:

  1. Click Add Reference Data
  2. Click the check box indicating the data you wish to display.

For a legend of the different boundaries click the Layer List icon at the bottom of the map, then click the dropdown arrow to the right of the layer checkbox. This will drop down with a legend indicating each data layer on your map. Visit the Layer List Topic to learn more about that mapping feature.
