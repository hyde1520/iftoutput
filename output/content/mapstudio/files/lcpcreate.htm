Creating a Landscape

In IFTDSS, a landscape refers to a spatial file that contains elevation data and vegetation/fuels data in .lcp file format.

You may generate a landscape by downloading LANDFIRE data.

In future releases of IFTDSS you will be able to upload your own .lcp files

Downloading a Landscape File, or generating your landscape, will allow you to run several of the models within IFTDSS. This is because the landscape file contains the Fire Behavior Fuel Model information required for many operations.

Landscape Files can be created in Map Studio, or in the Landscape Evaluation Stage of the Planning Cycle under Generate Landscape Summary.

Once created, landscapes can be viewed in My Workspace by

  1. Selecting My Workspace from the Top Navigation menu
  2. Selecting the folder you chose to store your landscape
  3. Select the name of your landscape and either view a summary, or view your landscape on the map

Or in Map Studio by

  1. Selecting Map Studio from the Top Navigation menu
  2. Click the Landscape Tools icon
  3. Make sure the Add to Map tab is selected
  4. Select the Landscape you would like to view
