The Planning Cycle - Landscape Evaluation

The Landscape Evaluation stage links to tasks that allow you to define the current fuel conditions on your landscape and their associated risks. These tasks will guide you in creating and understanding a landscape as well as writing goals to reduce risk.

Specific Tasks include:

Generate a Landscape

Downloading a Landscape File, or generating your landscape, will allow you to run several of the models within IFTDSS. This is because the landscape file contains the Fire Behavior Fuel Model information required for many operations.

Landscape Files can be created in Map Studio, or in the Landscape Evaluation Stage of the Planning Cycle under Generate Landscape Summary.

Once created, landscapes can be viewed in My Workspace by

  1. Selecting My Workspace from the Top Navigation menu
  2. Selecting the folder you chose to store your landscape
  3. Select the name of your landscape and either view a summary, or view your landscape on the map

Or in Map Studio by

  1. Selecting Map Studio from the Top Navigation menu
  2. Click the Landscape Tools icon
  3. Make sure the Add to Map tab is selected
  4. Select the Landscape you would like to view

Calculate Landscape Fire Behavior

[This will be updated before first release - UIdaho Update]

To run fire behavior across a landscape

  1. Navigate to the playground tab
  2. Select Basic from the dropdown menu
  3. Select Run

You will be prompted to name your run, make sure to give it a descriptive name so it can be easily identified later.

If you have already created your landscape file, choose a the landscape file from the dropdown menu. If you have not yet created a landscape file, you may do so by selecting 'Create New Landscape'.

Note: Selecting Create New Landscape will allow you to select a new landscape from within the modeling interface, you will not have to leave the interface.

Input windspeed and direction in degrees (1-360).

Specify a crown fire calculation method and foliar moisture content (in percent)

If you have a weather file for fuel moisture conditioning, you may add that here. If not, you may skip this step.

Input the fuel moistures for 1, 10, and 100 hour fuels, as well as herbaceous and woody fuels. When all the required inputs are filled the Run button will turn green to inform you the model is ready to run. Select Run to run the fire behavior model.

When the fire behavior model has completed a run, you will automatically be taken back to the Playground page, where you may left click on your run to view or edit it.
