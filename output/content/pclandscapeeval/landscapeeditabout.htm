Editing Landscape Data for Landscape Evaluation

Getting Ready to Edit

Edit Landscape will walk you through editing your landscape based on a default set of rules you select, or a custom set of rules you create.

First, select a landscape to edit from the Starting Landscape dropdown menu.

Notice you may access the Landscape rules by either clicking on the + icons under the landscape dropdown menu, or select the dropdown headings . As you select or create rules, they will be listed in the Edit Landscape section of the page.

To split your display to enable a map view of your landscape on the left side of the screen, and the landscape edit interface on the right, select the globe icon at the top left of the page.

Add Default Fuels Treatment/Disturbance Rule

There are four default treatment or disturbance types from which you may apply to your landscape.

In the above image, we have selected a rule that states a 20% thin of all species and all size classes, followed by a pile burn, which occurred 2-5 years ago. The rule applies only to the area selected by the Landscape Mask "Treatment Unit MM_2017_01".

Specify the treatment or disturbance type, and when prompted, select additional information about that treatment/disturbance type.

Most of these rules will require you to select additional information about the treatment/disturbance type. Options for the rules include:

  • Thin: Slash Removed
    • Thin 20% of all species for all size classes: Pile burn
    • Thin 65% of all species for all size classes: Pile burn
  • Thin: Slash Remains
    • Thin 55% of all material <6” DBH: lop and scatter
    • Thin 75% of all material <6” DBH; masticate ground fuels
    • Thin 90% of all material <8” DBH; masticate ground fuels
  • Clear Cut
    • Clearcut and Broadcast Burn
  • Wildland Fire
    • Low severity fire
    • Moderate severity fire
    • High severity fire

You must then specify a time since the treatment or disturbance

  • 1 year
  • 2-5 years
  • 5-10 years

You have the option of defining a Landscape Mask to apply your rules to.

Once you have set your rule, select Add to Rules. You may add as many rules as necessary.

If you begin creating editing rules, and navigate away from the edit page without saving, IFTDSS will retain those rules if you return, or begin edits through another part of the system (ie leaving Landscape Evaluation edits and then going into landscape comparison and editing). Partially completed edits will persist until they are saved or deleted.

 

For specific detail on what each default rule specifically describes, and the LANDFIRE disturbance code associated with it, visit the Treatment Rule Background Topic.

Add Landscape Edit Rule

You may also create your own landscape edit rules by selecting specific conditions, and how you would like to modify the landscape in instances where those conditions occur.

In the image above we have created a rule that finds any canopy base height values less than 0.5 meters that occur within Fuel Model TU3, and change those values to 1.0 meters. Because we did not input a Landscape mask, this rule will apply to our entire landscape.

Where these conditions exist:

Use this section to define the attributes of the landscape you would like to edit:

  • Attribute: Define the attribute you wish to edit.
  • Operator:
    • is equal to
    • is less than
    • is greater than
    • is less than or equal to
    • is greater than or equal to
    • has no assigned value
  • Value: The value you want to operator to apply to

Modify the following values:

Use this section to specify how you want the defined landscape attributes to be edited:

  • Attribute: The attribute you wish to edit
  • Modifier:
    • Set to
    • Clamp to a minimum
    • Clamp to a maximum
    • Multiply by
    • Increase by
    • Decrease by
    • Clear value (set to no data)
  • Value: The value that the modifier applies to.

Once you have set your rule, select Add to Rules. You may add as many rules as necessary.

Applying Landscape Edits

Once you rules are selected, give the new edited a landscape a descriptive and select Create New Landscape.

Edited landscapes will be created as new files, this ensures you will always have the original landscape for comparison

Edited landscapes are available in My Workspace, stored by default in the same folder as the landscape you are making edits to

When clicking on landscape files that have been edited in My Workspace, you will see an option to View Edit Rules. This displays the edits that were made on that landscape.
