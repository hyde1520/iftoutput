Auto97 Summary

The Auto97 Summary feature creates a fire behavior summary based on the highest 97th percentile of fire weather. IFTDSS uses your landscape file and automatically gathers 97th percentile weather information from the nearest Remote Automatic Weather Station (RAWS) to run fire behavior and summarize the outputs.

Inputs

Fuels

Inputs are automatically drawn from your Landscape file and include:

  • Fuel Model
  • Canopy Cover (percent)
  • Stand Height (meters)
  • Canopy Base Height (meters)
  • Canopy Bulk Density (kg/m3)
  • Aspect (degrees)
  • Slope (degrees)
  • Elevation (meters)

Weather (fuel moistures)

Uses average 97th percentile Energy Release Component (ERC-G) live and dead fuel moisture contents from the closest RAWS. Closest is determined using the algorithm from the Wildland Fire Decision Support System (WFDSS) which minimizes horizontal and vertical distances.

Winds

The month of the peak ERC-G plus the months before and after that date (three total months) are summarized into a table. Table cells contain the percentage of observations for the normal burning period (1200-2000 hrs) as follows:

Wind speed and direction is selected from the table as follows:

  • The row with 97th percentile speed is identified (13-18 in the above example),
  • the mid-point wind speed (15 mph in this case) is converted to 1 minute average (20 mph using Table 1 from Crosby and Chandler 2004),
  • the cell with the largest value for that row is used as the direction (west in this example),
  • the resultant wind speed and direction is used as input for Wind Ninja to create the wind grids used in the run.

Outputs

Outputs are displayed on a map across the landscape, as a bar chart, pie chart, and in tabular form. The summary report will include both the inputs above and the following fire behavior outputs:

  • Flame Length (meters)
  • Spread Rate (m/min)
  • Intensity kW/m)
  • Heat Area (kJ/m2)
  • Crown State (crown fire activity, No Fire, Surface Fire, Passive Fire)

Exporting

Each set of outputs may be printed, or saved to your computer for use outside of IFTDSS if desired.
