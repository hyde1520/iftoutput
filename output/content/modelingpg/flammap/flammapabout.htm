Fire Behavior Basic - Driven by FlamMap 5.0

 

Read a Step by step walk- through with images

To run fire behavior across a landscape

  1. Navigate to the playground tab
  2. Select Basic from the dropdown menu
  3. Select Run

You will be prompted to name your run, make sure to give it a descriptive name so it can be easily identified later.

If you have already created your landscape file, choose a the landscape file from the dropdown menu. If you have not yet created a landscape file, you may do so by selecting 'Create New Landscape'.

Note: Selecting Create New Landscape will allow you to select a new landscape from within the modeling interface, you will not have to leave the interface.

Input windspeed and direction in degrees (1-360).

Specify a crown fire calculation method and foliar moisture content (in percent)

If you have a weather file for fuel moisture conditioning, you may add that here. If not, you may skip this step.

Input the fuel moistures for 1, 10, and 100 hour fuels, as well as herbaceous and woody fuels. When all the required inputs are filled the Run button will turn green to inform you the model is ready to run. Select Run to run the fire behavior model.

When the fire behavior model has completed a run, you will automatically be taken back to the Playground page, where you may left click on your run to view or edit it.
